# 3. Context and Scope

## Context

ConSurv as camera-organization-system should be usable for everyone and every organisation in a self-hosted way.

## Scope

The release-artifact is a deployable only as OCI-image.
The user must deploy ConSurv on his own system.

There is no official cloud-service which provides ConSurv.

ConSurv does not provide any TLS-certificate.
The user must obtain his own domain and TLS-certificates.
