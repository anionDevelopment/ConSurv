# Hints

## Requirements

The following tools from the [tools-list](https://github.com/anionDev/ScriptCollection/blob/main/ScriptCollection/Other/Reference/ReferenceContent/Articles/RequirementsForCommonProjectStructure.md#Tools) are required to build this code-unit:

- `cyclonedx-npm`
- `docfx`
- `git`
- `gitversion`
- `ng`
- `npm`
- `openapi-generator-cli`
- `python`
- `reportgenerator`
- `scriptcollection`

## IDE

The recommended IDE for this codeunit is [Visual Studio Code](https://code.visualstudio.com/).

Start the project using `task RunFrontend` (or shorter: `task rf`).
This starts the project technically without an attached debugger.
You can use the developer-tools of your browser to debug the webapplication.
Alternatively you can run the frontend not using the `RunFrontend`-button but with the debug-button of your frontend-IDE instead.
